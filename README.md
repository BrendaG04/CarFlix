# 🏎️ CarFlix  

CarFlix is a full-stack catalog of over **1,000 popular car makes and models** from **2024 & 2025**, providing details such as **engine type, seating capacity, and estimated price**.  

Built using **Spring Boot**, **ReactJS**, and **PostgreSQL**, this project helps users efficiently browse and explore different cars.  


## 📌 Features  

- ✅ Browse car makes and models  
- ✅ View details such as **engine type, seating capacity, and price**  
- ✅ **PostgreSQL** database integration  
- ✅ CSV file import for **bulk data entry** (Data from [NHTSA](https://www.nhtsa.gov))  
- ✅ REST API built with **Spring Boot**
- ✅ Frontend development with **ReactJs** 

## 🚀 Tech Stack  

- **Backend:** Java, Spring Boot, Hibernate, SQL  
- **Frontend:** Javascript, HTML, CSS, ReactJs
- **Authorization** JWT
- **Database:** PostgreSQL  
- **Version Control:** Git, GitHub  
- **Tools:** Excel  

## 📂 API Endpoints  

| Method | Endpoint           | Description              |
|--------|-------------------|--------------------------|
| **GET**    | `/api/cars`       | Get all cars              |
| **POST**   | `/api/cars`       | Add a new car             |
| **PUT**    | `/api/cars/{id}`  | Update car details        |
| **DELETE** | `/api/cars/delete/{id}`  | Delete a car              |
